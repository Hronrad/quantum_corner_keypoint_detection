QNN Improvement Demo

当前目标不是宣称量子优势，而是展示 QNN 结构改进、噪声鲁棒性与消融分析。MLP 仍是 clean 条件下最强 baseline；salt-and-pepper 全量验证中 QNN 的固定阈值 F1 更高，但 PR-AUC 仍低于 MLP。

指标解释

MetricMeaning
Precision预测为 keypoint 的点或 patch 中有多少是真的，低值表示误检多。
RecallGT keypoint 中有多少被找到了，低值表示漏检多。
F1Precision 与 Recall 的调和平均，当前阶段作为主指标。
PR-AUC跨阈值的 Precision-Recall 曲线面积，反映概率或响应分数的整体排序能力。

ORB 结果解释

ORB 主要面向自然图像中的可重复纹理点，检测依赖 FAST，描述依赖旋转 BRIEF。本任务的图像是稀疏几何线条，边缘和抗锯齿像素会产生大量额外响应；评估只把几何交点算作 GT，因此 ORB 召回高但 false positive 极多，Precision 与 F1 很低。

结论

  • 8-D 特征、Z+ZZ readout、trainable input scaling 已接入并完成训练。
  • 消融覆盖 L=1/2/3、none/linear/ring entanglement、Z vs Z+ZZ readout。
  • 噪声实验覆盖 clean、Gaussian、blur、salt-and-pepper，并补充了 salt-and-pepper 全测试集验证。
  • 后续重点应放在多随机种子验证、更多 QNN 训练样本、hard negative mining 与 keypoint type classification。
